# Social HQ

Hub operativo unico per la crescita social personale e brand di Domenico Dentamaro / 2D Sviluppo Immobiliare.

## Struttura

- strategy/: strategia, audit, posizionamento, regole editoriali
- analytics/: screenshot, note insight, report performance
- assets/photos/: foto personali, contesto lavoro, territorio
- copy/: post pronti, risposte ai commenti, bio, hook
- calendar/: piano editoriale giornaliero e settimanale

## Regola di lavoro

1. Tutto il materiale social entra qui.
2. Le foto vanno in assets/photos/ con nomi chiari.
3. Gli screenshot insight vanno in analytics/.
4. I post pronti vanno in copy/.
5. I piani approvati vanno in calendar/.

## Naming consigliato

- foto: YYYY-MM-DD-descrizione.jpg
- insight: YYYY-MM-DD-canale-note.md oppure screenshot originale
- copy: YYYY-MM-DD-canale-topic.md
- piano: YYYY-MM-DD-week-plan.md
